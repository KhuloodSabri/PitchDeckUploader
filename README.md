# PitchDeckUploader

## How to Run
- Clone the repository
- On your machine dowload and install:
  - nodejs
  - graphicsmagic
  - ghostscrip
  - libreoffice
- Go to the directory where you cloned the project, and run these commands:
  - npm i
  - node src/app.js
- Then, you should be able to visit the website on localhost:3000
